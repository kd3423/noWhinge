Why noWhinge?





Multiple departments of the government have separate web complaint portals. This can be hassle as the users have to figure out which government branch to contact, and within the branch which department. We plan to collate all forms in one portal, so that users can use a single platform for different grevencies. The UI for most portals are different and are not intuitive, we plan to make an easy to use interface. The data gathered will be analysed and problem based analysis of the region will be made public on the website - this will be the locality based review. We plan to make the review process user interactive. Lastly we plan to link social media to the problems so that they can gain limelight and won't be ignored.
